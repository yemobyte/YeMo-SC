POST /api/screenshot

Initiate a screenshot capture sequence. The server renders the target URL via a headless browser engine.

Parameter Type Required Description
url String YES Absolute URL (e.g., https://google.com or http://1.1.1.1)
deviceType String NO Enum: desktop, macbook-pro, iphone-14, pixel-7, custom
customWidth Integer CONDITIONAL Width in px. Required if deviceType is "custom".
customHeight Integer CONDITIONAL Height in px. Required if deviceType is "custom".

Implementation Examples

Javascript (Es6)

const captureScreen = async () => {
  const url = '/api/screenshot';
  const payload = {
    url: 'https://example.com',
    deviceType: 'desktop'
  };

  try {
    const response = await fetch(url, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(payload)
    });

    const result = await response.json();
    console.log(result);
  } catch (error) {
    console.error('API Error:', error);
  }
};

Python 3

import requests

endpoint = "http://your-app.com/api/screenshot"
data = {
    "url": "https://example.com",
    "deviceType": "macbook-pro"
}

try:
    response = requests.post(endpoint, json=data)
    result = response.json()
    print(result)
except Exception as e:
    print(f"Request Failed: {e}")

cURL

curl -X POST http://your-app.com/api/screenshot \
  -H "Content-Type: application/json" \
  -d '{
    "url":"https://google.com", 
    "deviceType":"desktop"
}'
