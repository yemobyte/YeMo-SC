API Reference

Native HTTP Integration

POST /api/screenshot

Captures a screenshot of any URL or IP address. Supports standard or high-resolution archiving.

Parameter Type Required Description
url String YES Full URL or IP (e.g., http://1.1.1.1)
deviceType String NO desktop, iphone-14, pixel-7, custom
customWidth Int COND Required if "custom"
customHeight Int COND Required if "custom"

Usage Examples

cURL

curl -X POST http://your-app/api/screenshot \
-H "Content-Type: application/json" \
-d '{"url":"http://157.230.33.80:3000"}'

Fetch API

fetch('/api/screenshot', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    url: 'http://example.com'
  })
});
