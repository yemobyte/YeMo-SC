POST /api/screenshot

Initiate a screenshot capture sequence via headless engine. Supports HTTP, HTTPS, and IP addresses.

Parameter Type Required Description
url String YES Absolute URL or IP address
deviceType String NO desktop, macbook-pro, iphone-14, pixel-7, custom
customWidth Integer COND Required if deviceType is "custom"
customHeight Integer COND Required if deviceType is "custom"

Implementation Examples

Javascript

const capture = async () => {
  const response = await fetch('/api/screenshot', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ url: 'http://157.230.33.80:3000', deviceType: 'desktop' })
  });
  console.log(await response.json());
};

Python

import requests
data = {"url": "https://google.com", "deviceType": "macbook-pro"}
res = requests.post("http://your-app/api/screenshot", json=data)
print(res.json())

cURL

curl -X POST http://your-app/api/screenshot \
  -H "Content-Type: application/json" \
  -d '{"url":"http://157.230.33.80:3000", "deviceType":"custom", "customWidth":1920, "customHeight":1080}'
