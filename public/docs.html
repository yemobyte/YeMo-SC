POST /api/screenshot

Initiate a screenshot capture sequence. The server renders the target URL via a headless browser engine.

Parameter Type Required Description
url String YES Absolute URL (e.g., https://google.com)
deviceType String NO Enum: desktop, macbook-pro, iphone-14, pixel-7, custom
customWidth Integer CONDITIONAL Width in pixels. Required if deviceType is "custom".
customHeight Integer CONDITIONAL Height in pixels. Required if deviceType is "custom".

Implementation Examples

Javascript (Es6)

const captureScreen = async () => {
  const url = 'http://localhost:3000/api/screenshot';
  const payload = {
    url: 'https://example.com',
    deviceType: 'custom',
    customWidth: 1080,
    customHeight: 1350
  };

  try {
    const response = await fetch(url, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(payload)
    });

    const result = await response.json();
    console.log(result);
  } catch (error) {
    console.error('API Error:', error);
  }
};

Python 3

import requests

endpoint = "http://localhost:3000/api/screenshot"
data = {
    "url": "https://example.com",
    "deviceType": "custom",
    "customWidth": 1200,
    "customHeight": 800
}

try:
    response = requests.post(endpoint, json=data)
    result = response.json()
    
    if result.get("status"):
        print(f"Image saved at: {result['data']['url']}")
    else:
        print(f"Error: {result.get('message')}")
        
except Exception as e:
    print(f"Request Failed: {e}")
